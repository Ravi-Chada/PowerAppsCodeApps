# Customer Feedback Assistant

## Scenario
Build a Code App for collecting customer feedback, summarizing themes, and assigning follow-up actions.

## Power Platform Components
- **Power Apps**: Feedback submission and review UI
- **Power Automate**: Trigger flow when new feedback is submitted
- **Dataverse**: Store feedback records and action items
- **Copilot Studio (optional)**: Draft suggested response text

## Suggested Extensions
- Add sentiment scoring and escalation thresholds
- Add dashboard views for trend tracking
